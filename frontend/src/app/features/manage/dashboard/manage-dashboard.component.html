Как загружать фото

  • Товары — главное фото карточки + галерея ракурсов. Кнопка «Загрузить фото», не нужно вручную копировать ссылки.
  • Категории и портфолио — одно понятное фото на карточку; превью видно сразу.
  • Страницы — у каждого блока подсказка, какое фото туда класть (герой, СВО, о компании…).
